매출관리 AI 직원

Commerce AI Company · 운영팀 · 담당업무: 매출·광고비 데이터 분석 및 리포트

전체보기는 모든 플랫폼 탭의 데이터를 합산해서 보여줍니다. 새 데이터를 추가하려면 플랫폼 탭을 선택하거나 "+ 플랫폼 추가"로 새 탭을 만드세요.

📥 정산/매출 데이터 불러오기

쿠팡 Wing → 정산 → 매출내역 다운로드(CSV) 파일을 그대로 올리면 됩니다.
🔌 실제 쿠팡 매출 데이터 연동 (로컬 서버 실행 시)

같은 폴더의 server.js를 node server.js로 실행하면(사전에 .env에 쿠팡 Open API 키 입력) 아래에서 실제 매출 데이터를 바로 가져올 수 있습니다.

쿠팡 주문 데이터에는 취소/반품 건도 함께 들어있는 경우가 많습니다. 주문상태 컬럼을 지정하면 해당 건은 매출 집계에서 자동으로 제외되어 매출이 부풀려지지 않습니다.

📢 광고비 데이터 불러오기 (선택 · ROAS 계산용)

날짜별 집행 광고비만 있으면 됩니다. 채널이 여러 개면 합계를 한 파일로 만들어 올려도 됩니다.

상품명 컬럼을 지정하면 품목별로 광고를 켠 달/끈 달을 구분해 효과를 비교합니다. 지정하지 않으면 전체 합계로만 ROAS를 계산합니다. 매출 데이터를 먼저 불러온 뒤 광고비를 적용해주세요.

🏷️ 원가 데이터 불러오기 (선택 · 이익 계산용)

상품명과 개당 원가만 있으면 됩니다. 상품별로 1행씩만 있으면 되고, 날짜는 필요 없습니다.

매출 데이터의 상품명과 정확히 일치해야 매칭됩니다. 매출 데이터를 먼저 불러온 뒤 원가를 적용해주세요.

고급 설정 · Claude API로 AI 코멘트 강화 (선택)

키가 없어도 규칙 기반 AI 코멘트로 전체 흐름을 체험할 수 있습니다. 쿠팡 Open API를 코드로 직접 연동하는 방법은 같은 폴더의 README.md를 참고하세요.
