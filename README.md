# 매출관리 AI 직원 — Commerce AI Company

전자상거래 사업자를 위한 **매출관리** 도구입니다. 단순히 매출액을 집계하는 것을 넘어,
① 취소/반품을 제외한 정확한 매출, ② 상품별 추세와 이상 탐지, ③ 광고비 대비
매출 효율(ROAS), ④ 원가를 반영한 실제 손익까지 함께 봐야 "잘 팔리고 있는지"가
아니라 "돈이 남는 장사인지"를 판단할 수 있다는 문제의식에서, 광고비와 원가를
매출 데이터에 선택 컬럼으로 함께 넣어 같이 분석하는 기능을 포함합니다(광고비·원가
전용 파일을 따로 만들 필요가 없습니다).

특히 광고비는 품목(상품)까지 매핑하면 **품목별로 광고를 켠 달과 끈 달의 평균
매출을 비교**해 "이 상품은 광고를 계속 돌릴 가치가 있는지, 꺼도 잘 팔리는
자연 판매 상품인지"를 월 단위로 보여줍니다.

## 플랫폼 탭 (쿠팡/네이버 등 소스별로 데이터 분리 관리)
화면 상단에 "전체 / 쿠팡 / 네이버 / + 플랫폼 추가" 탭이 있습니다. 각 탭은 서로 독립된
매출·광고비·원가 데이터를 갖고 있고, 브라우저에 저장되어 새로고침해도 유지됩니다.

- 탭을 눌러 전환하면 그 플랫폼에 업로드했던 데이터와 리포트가 그대로 나타납니다.
- "+ 플랫폼 추가"로 옥션, 11번가, 자사몰 등 원하는 이름의 탭을 계속 만들 수 있습니다.
- **"전체"** 탭은 모든 플랫폼 탭의 데이터를 합산해서 보여주는 계산 결과이며, 직접
  업로드는 할 수 없습니다(각 플랫폼 탭에서 업로드하세요). "🏆 상품별 매출 순위"와
  "📅 품목별 월별 상세"는 전체 탭에서도 그대로 동작하므로, 플랫폼을 합쳐서 품목 기준으로
  보고 싶을 때는 전체 탭을, 특정 플랫폼만 보고 싶을 때는 해당 탭을 선택하면 됩니다.
- 각 플랫폼 탭 옆의 👁/🚫 버튼은 **비활성화**입니다 — 데이터는 그대로 두고 "전체" 탭
  집계에서만 빼는 것으로, 탭 자체는 계속 눌러서 쓸 수 있습니다. × 버튼은 **삭제**로,
  그 플랫폼 탭과 저장된 매출·광고비·원가 데이터를 완전히 지웁니다(되돌릴 수 없어
  확인 후 진행). 최소 1개 플랫폼은 항상 남아 있어야 합니다.

## 데이터 검색 (키워드 칩) · 개별 데이터 ON/OFF·수정·삭제
"🔍 데이터 검색" 카드는 불러온 매출 데이터 전체를 항상 표로 보여줍니다(검색해야만
나타나지 않습니다). 검색어를 입력하면 품목명·구매자명·날짜와 일치하는 항목만
좁혀지고(↑/↓로 이동, Enter로 선택), 클릭(또는 Enter)하면 작은 칩으로 남아 결과를
필터링합니다. 칩은 다시 클릭하면 제거됩니다. 항목은 색으로만 구분됩니다(품목=파랑,
구매자=초록, 날짜=노랑, 플랫폼=보라). 구매자명을 검색하려면 매출 데이터 매핑에서
"구매자명/회원명 컬럼"을 지정해야 합니다(선택 사항).

"전체" 탭에서는 각 매출 라인이 어느 플랫폼(쿠팡/네이버 등)에서 왔는지 결과 표에
"플랫폼" 열로 표시되고, 플랫폼 이름도 칩으로 선택해 필터링할 수 있습니다.

검색 결과 각 행에는 **ON/OFF · 수정 · 🗑 삭제** 버튼이 있습니다.

- **ON/OFF**는 그 매출 한 건을 통째로 켜고 끄는 버튼입니다. OFF로 꺼두면 삭제하지 않고도
  총 매출·상품 순위·품목별 월별 상세 등 **모든 계산에서 제외**됩니다(테스트성 주문이나
  의심스러운 건을 지우지 않고 잠시 빼둘 때 유용합니다). 다시 누르면 즉시 계산에 포함됩니다.
- **수정**을 누르면 그 행이 바로 그 자리에서 입력 칸으로 바뀝니다(다른 곳으로 스크롤되지
  않습니다). 값을 고치고 "저장"을 누르면 바로 반영되고, "취소"를 누르면 원래 값으로
  되돌아갑니다.
- **🗑**은 그 한 건을 완전히 삭제합니다(되돌릴 수 없음, 확인 후 진행).
- 검색창 위 **"전체 보기 / ON만 보기 / OFF만 보기"** 버튼을 누르면 검색어를 입력하지
  않아도 켜져 있는 데이터 또는 꺼둔 데이터만 목록으로 볼 수 있습니다.
데이터를 통째로 지우려면 "📥 데이터 불러오기"의 파일 목록에서 해당 파일을 🗑로 삭제하거나
(아래 "불러온 파일 목록" 참고), 상단 플랫폼 탭의 × 버튼으로 탭 자체를 지우세요.

## 광고 ON/OFF 직접 지정
"📅 품목별 월별 상세"의 각 월별 행에는 "미지정 / ON / OFF" 버튼이 있습니다. 클릭할
때마다 순서대로 바뀌며, 실제 매출 데이터를 기준으로 ON인 달과 OFF인 달의 **평균
매출을 그대로 비교**해 보여줍니다(미래 변동을 예측하지 않고, 실제로 있었던 숫자만
사용합니다). 광고비 CSV로 자동 판별된 상태 위에 직접 지정한 값이 우선 적용되므로,
광고비 데이터가 없는 품목도 이 버튼만으로 ON/OFF 비교를 만들 수 있습니다.

## 관리 기능 (지속 사용을 위한 판단 도구)
- **💾 전체 백업/복원** — 화면 상단 버튼으로 모든 플랫폼의 데이터·설정을 JSON 파일 하나로
  내려받고, 다른 브라우저/PC에서 그 파일을 올리면 그대로 복원됩니다. 데이터가 브라우저에만
  저장되므로 주기적으로 백업해 두세요.
- **🎯 월 매출 목표** — 플랫폼별 월 목표 금액을 설정하면 매출 KPI 아래에 달성률 게이지와
  "이 속도면 월말 약 ○○원" 페이스 예측이 표시됩니다(전체 탭은 플랫폼 목표의 합산).
- **📌 오늘의 판단** — 데이터를 행동 제안으로 바꿔 최대 3개까지 보여줍니다: ① 역마진(원가·
  광고비 반영 시 손해나는 상품), ② 광고 손익분기(상품별 마진율로 계산한 손익분기 ROAS
  미달 시 중단 검토 + 월 절감액), ③ 자연판매(광고 OFF 달에도 매출이 유지되는 상품),
  ④ 급성장/급감(재고 확보·원인 확인). 같은 판정이 상품 순위와 월별 상세에 배지로도 붙습니다.
- **👥 재구매 고객** — 구매자명을 매핑하면 재구매율, 재구매 고객 매출 비중, VIP 10명이
  표시됩니다.
- **📆 요일별 매출 패턴** — 하루 평균 매출 기준으로 요일별 강세를 비교해 광고·프로모션
  집행 요일을 정하는 데 참고할 수 있습니다.

## 실행 방법
`index.html`을 더블클릭해서 브라우저로 열면 바로 사용할 수 있습니다. 설치나 회원가입,
인터넷 연결이 필요 없고 데이터는 브라우저 localStorage에만 저장됩니다(다른 사람에게
전송되지 않습니다).

## 실제 내 쇼핑몰 데이터로 쓰는 방법
1. 쿠팡 Wing → 정산 → 매출내역/정산내역 다운로드 (CSV 또는 엑셀)
2. index.html에서 파일 업로드
3. 열 매핑 확인(날짜·상품명·매출액은 필수, 원가·광고비·채널 등은 이 파일에 있으면만
   채우면 됨) → "데이터 추가"

같은 폴더의 `쿠팡주문.csv`는 실제 쿠팡 주문 다운로드 파일과 같은 구조(주문일자, 주문번호,
구매자ID, 상품명, 옵션, 수량, 판매가, 주문상태)의 샘플입니다.

- 주문상태에 취소/반품이 섞여 있으면 "주문상태 컬럼"을 지정하세요. 취소/반품/환불/교환/
  구매취소/철회 및 영문(cancel/refund/return/exchange, 대소문자 무관)이 포함된 상태값은
  자동으로 매출 집계에서 빠집니다(별도 설정 불필요). 단, 아직 확정되지 않았거나
  (요청/신청/접수) 무산된(거부/거절/반려/불가/실패) 취소·반품, 그리고 "취소 철회"처럼
  취소가 되돌려진 상태는 실제 판매가 유지된 것으로 보고 매출에 포함합니다.
- 이 샘플의 "판매가"는 주문 라인의 합계 금액이라 매출액 컬럼으로 바로 매핑하면 됩니다.
  판매가가 단가인 파일은 수량 컬럼도 함께 지정해 검산해보세요.

#### 업로드는 한 곳, 매출 파일 하나에 원가·광고비도 함께 넣으면 됩니다
"📥 데이터 불러오기" 카드 하나로 매출 데이터를 불러옵니다. 파일을 올리면 컬럼 매핑
화면 하나에 날짜·상품명(필수) + 매출액·수량·주문상태·구매자 + 원가 + 광고비·채널
항목이 함께 나옵니다. 원가·광고비·채널은 모두 **선택 컬럼**이라 이 파일에 있는 것만
채우면 되고, 비어 있어도 매출 데이터는 정상적으로 저장됩니다(원가·광고비가 없는 상품/
날짜는 손익·ROAS 계산에서 자동으로 제외되거나 0으로 표시됨). 나중에 값을 알게 되면
"🔍 데이터 검색"에서 해당 건을 찾아 그 자리에서 채워 넣거나 수정할 수 있습니다.
"데이터 추가"를 누르면 매핑한 컬럼이 모두 같은 매출 레코드에 반영됩니다.

CSV와 엑셀(.xlsx) 모두 지원합니다(구버전 `.xls`는 미지원, "다른 이름으로 저장 → .xlsx"로
변환). 컬럼 순서는 상관없이 매핑 화면에서 역할만 지정하면 됩니다. CSV 파일의 인코딩
(UTF-8 또는 한글 엑셀에서 흔한 EUC-KR/CP949)은 자동으로 감지해서 읽으므로, 컬럼 매핑
화면에 한글이 깨져 보이면 파일을 다시 올려보세요.

#### 포맷 프리셋 (API 키 불필요)
컬럼 매핑 화면 하나에 프리셋 드롭다운이 하나만 있어, 저장된 컬럼명과 실제 파일 헤더가
일치하면 매출·광고비·원가 컬럼이 한 번에 자동으로 매핑됩니다. 일치하지 않는 항목은
자동으로 채워지지 않을 뿐(잘못된 값이 채워지지는 않음) 직접 골라 "프리셋으로 저장"을
누르면 다음부터 재사용할 수 있습니다. 더 이상 쓰지 않는 프리셋은 🗑로 삭제할 수 있습니다
(기본 제공 프리셋 제외).

- 매출: "쿠팡 (Wing 주문/매출내역)"(`쿠팡주문.csv` 구조로 검증됨), "네이버 스마트스토어
  (추정)"
- 광고비 컬럼명: "쿠팡 광고관리자 (추정)" · "네이버 검색광고 (추정)" · "메타 광고관리자
  (추정)" — 광고비·채널 컬럼명만 담고 있어, 매출 파일에 이 컬럼이 함께 있을 때 빠르게
  지정하는 용도입니다.
- 원가 컬럼명: "기본 (원가)" · "매입가 기준" · "제조원가 기준" — 원가 컬럼명만 담고
  있어, 매출 파일에 이 컬럼이 함께 있을 때 빠르게 지정하는 용도입니다.

필수 컬럼이 프리셋으로 모두 채워지면 컬럼 매핑 영역은 자동으로 접힙니다.

#### 파일은 계속 추가할 수 있습니다 (교체가 아니라 누적)
파일을 올려 "데이터 추가"를 누르면 기존 데이터 위에 더해집니다. 월별로 나뉜 파일을
순서대로 하나씩 올리면 됩니다. 파일마다 다른 프리셋을 써도 됩니다.

#### 불러온 파일 목록: 클릭해서 그 파일만 보기 + 재매핑 · 숨기기 · 삭제
"📥 데이터 불러오기" 제목 옆 숫자는 지금까지 쌓인 파일(또는 수기 입력 묶음) 개수입니다.
파일명을 **클릭**하면 오른쪽 화면(매출 KPI·데이터 검색·광고비 & ROAS·원가 & 손익·
상품 순위·월별 상세) 전체가 그 파일의 데이터만으로 좁혀지고, 동시에 열 매핑 화면도
다시 열려 컬럼을 고쳐서 다시 적용할 수 있습니다(기존에 그 파일이 만들어낸 데이터는
새 매핑 결과로 교체됩니다). 같은 파일명을 다시 클릭하면 좁혀보기가 풀리고 전체 데이터로
돌아갑니다. 옆의 **👁/🚫**는 그 파일의 데이터를 지우지 않고 계산에서만 뺐다 켰다 할 수
있고, **×**는 그 파일이 만든 매출(원가·광고비 포함) 데이터를 오른쪽 화면에서도 통째로
삭제합니다(되돌릴 수 없음). 업로드할 때 원본 표를 브라우저에 함께 저장해 두므로,
새로고침한 뒤에도 파일명을 클릭하면 컬럼 매핑을 다시 열어 고칠 수 있습니다(아주 큰
파일은 용량 문제로 저장이 생략될 수 있고, 그 경우 ×로 지우고 새로 올려주세요).

파일 없이 "직접 입력"으로 추가한 매출 데이터(원가·광고비 포함)는 모두 **"수기 입력
목록"**이라는 이름 하나로 파일 목록에 묶여서 나타나며, 다른 파일과 똑같이 클릭해서
좁혀보기·숨기기·삭제할 수 있습니다.

#### 샘플 양식 다운로드
카드 아래 작은 "매출 양식" 버튼으로 날짜·상품명·매출액·개당 단가·수량·구매자·원가·
광고비·채널 컬럼이 모두 담긴 빈 CSV를 내려받아 참고하거나 채워서 그대로 업로드할 수
있습니다. 금액은 **매출액(합계)과 개당 단가 중 하나만** 채우면 됩니다 — 개당 단가를
채웠다면 컬럼 매핑에서 "개당 단가" 컬럼을 매출액으로 선택하세요(컬럼명에 "단가"가
들어 있으면 [합계 금액|개당 단가] 토글이 자동으로 전환되고, 단가 × 수량으로 계산됩니다).
원가·광고비·채널 컬럼은 비워둬도 되며, 나중에 "🔍 데이터 검색"에서 건별로 채워 넣을
수 있습니다.

#### 파일 없이 직접 입력
"파일 없이 직접 입력해서 추가"에는 매출/광고비/원가를 유형으로 나누지 않고 날짜·상품명·
매출액·수량·구매자·원가·광고비·채널 입력칸이 한 폼에 모두 열려 있습니다. 필수(날짜·
상품명·매출액)만 채우고 나머지는 비워둬도 됩니다. 추가한 데이터는 파일 목록의 "수기
입력 목록" 항목으로 모이며(위 항목 참고), "🔍 데이터 검색" 결과 목록에서 그 자리에서
바로 수정·삭제·ON/OFF할 수 있습니다.

#### 광고비 · ROAS
매출 파일에 광고비(선택) / 채널(선택) 컬럼을 함께 매핑하면(날짜·상품명은 위 공통 항목
사용) 광고비 합계, ROAS(매출÷광고비), 광고비율, 광고비 차감 순매출이 계산됩니다. ROAS가
1배 미만이거나 직전 기간 대비 크게 떨어지면 경고가 표시됩니다.

광고비가 입력된 매출 라인은 이미 상품명·날짜를 갖고 있으므로 "📅 품목별 월별 상세"에서
자동으로 품목별 광고 ON/OFF 구간의 평균 매출을 비교합니다. 값이 마이너스(지출을 음수로
표기하는 회계 방식)로 들어있어도 자동으로 절댓값으로 바꿔 계산합니다.

#### 원가 · 손익
매출 파일에 개당 원가(선택) 컬럼을 함께 매핑하면 총 원가, 매출총이익(매출-원가),
영업이익(매출총이익-광고비), 영업이익률이 계산됩니다. 원가가 입력되지 않은 라인은
0원으로 계산되고 리포트에 몇 개 품목이 빠졌는지 표시됩니다. 원가도 마이너스로 들어있으면
자동으로 절댓값으로 바꿔 계산합니다.

#### 상품별 매출 순위 (전체, 더보기로 펼쳐보기)
"🏆 상품별 매출 순위" 카드는 상위 10개만 보여주지 않고 **전체 품목을 순위대로** 계산합니다.
처음엔 10개만 보이고 "더보기"를 누르면 나머지가 모두 펼쳐집니다. Markdown 리포트에도 전체
순위가 그대로 나갑니다.

#### 품목별 월별 상세 (전체 품목, 펼쳐보기)
매출 데이터만 있어도 "📅 품목별 월별 상세" 카드에서 **순위와 무관하게 전체 품목**을 월별로
펼쳐볼 수 있습니다. 각 품목을 펼치면 월별 매출·원가·광고비·순이익·광고 ON/OFF 상태가
표에 나오고, 광고 ON 달과 OFF 달이 모두 있는 품목은 두 구간의 평균 매출을 자동으로
비교해줍니다.

## 처음 시작하는 순서
1. 상단 탭에서 사용할 플랫폼(쿠팡/네이버 등)을 선택
2. "📥 데이터 불러오기"에 파일을 올리거나 "파일 없이 직접 입력해서 추가"로 몇 건 입력 →
   프리셋이 있으면 선택, 없으면 컬럼을 직접 매핑 → "데이터 추가"
3. 파일이 더 있다면 계속 올리면 기존 데이터에 더해집니다(교체되지 않음). 잘못 올린 파일은
   목록에서 클릭해 재매핑하거나 🗑로 삭제
4. "📅 품목별 월별 상세"에서 품목을 펼쳐 월별 매출·원가·광고비·광고 ON/OFF 비교 확인,
   버튼으로 직접 ON/OFF 지정 가능
5. "🔍 데이터 검색"에서 특정 건을 찾아 그 자리에서 수정하거나, OFF로 꺼서 계산에서
   빼거나(삭제 아님), 🗑로 완전히 삭제
6. "네이버" 등 다른 플랫폼 탭으로 전환해 별도 데이터를 올리고, "전체" 탭에서 합산 결과 확인
7. handoff.json / Markdown 리포트 내보내기로 회의 자료·인수인계 문서를 바로 생성
